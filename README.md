# ChatBroSSO

Example Single Sign-On for chatBro.

### Development

Clone the repository
```shell
$ git clone https://github.com/SviatoIa/ChatBroSSO.git && cd ChatBroSSO
```

Install dependencies
```shell
$ npm install
```

Run
```shell
$ node chatbro_test.js
```

Test app listening on localhost:3000!
